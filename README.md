# Prisca Customer Service

#### Customer Service Android App for a 3D Printing Startup, the main goal of this app is to provide support and online ordering service to users.
#### There are two versions of this App, Users App (Prisca), and Customer Service App (PriscaCS), both apps are connected using firebase.
